# Angular

## The Angular Tutorial

1. Start by completing [Tour of Heros](https://angular.io/docs/ts/latest/tutorial/) tutorial. Then modify it to read from online data.

Install the angular shell
```sh
sudo npm install -g @angular/cli
cd ~
ng new angular-tour-of-heroes
cd angular-tour-of-heroes
npm install
ng serve --open
```

Open Atom and add *~/angular-tour-of-heroes* as a project to the side bar. Then open a browser an navigate to *http://localhost:4200*
* Note the files structure
